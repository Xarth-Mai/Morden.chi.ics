# Morden.chi.ics

提供 `.ics` 日历订阅文件，用于补充 Apple 设备日历缺失的节假日。

补充民间年轻群体广泛认同的国际节日。

## 已补充节假日列表

- **母亲节**：5月的第2个星期日
- **父亲节**：6月的第3个星期日
- **感恩节**：11月的第4个星期四
- **情人节**：2月14日
- **植树节**：3月12日
- **愚人节**：4月1日
- **教师节**：9月10日
- **万圣节**：10月31日
- **平安夜**：12月24日
- **圣诞节**：12月25日

## 食用方法
  
订阅地址：[Github Raw](https://raw.githubusercontent.com/Xarth-Mai/Morden.chi.ics/refs/heads/main/morden.ics)

1. 打开日历应用；
2. 点击应用下方 `日历` 字样，点击左下角添加日历；
3. 选择添加订阅日历，粘贴上订阅链接订阅即可。
